# geopy-world-cities
A GeoPy test running against a JSON file with all the most important world cities

### JSON format
```json
  {
    "country": "ES",
    "name": "Madrid",
    "lat": "40.4165",
    "lng": "-3.70256"
  }
```

### Example results
```

[...]

```
